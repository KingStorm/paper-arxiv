## Introduction
Paper Arxiv is a command line base academic paper management tool. It provide an efficient access to your pdf files, comments, tags and so on.
## Install
```
brew install pdftohtml
pip install pa
```
## Usage
```
pa init
pa add sj-98.pdf
pa find -a Tom
```
